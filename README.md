# do-a-doodle

like the popular vintage toy, draw with the help of two knobs that move your stylus horizontally or vertically

## how it works

1. turn the left knob to move the stylus left & right
2. turn the right knob to move it up & down
3. the stylus draws continuously, no lifting up the stylus
4. shake/click the button to clear the canvas
